# 📸 SnapAnnotator

**SnapAnnotator** is a lightweight real-time computer vision tool that captures webcam images and uses an AI vision model to generate structured, human-readable annotations.

It provides a clear scene description, detects objects, and overlays simple visual markers — making it ideal for learning, demos, and rapid prototyping.

---

## 🚀 Features

- 📷 **Real-time webcam capture**
- 🧠 **AI-powered image analysis (Moondream)**
- 📝 **Structured JSON output**
- 🟩 **Visual object annotations**
- 💬 **Interactive follow-up Q&A**
- ⚡ **Optimized for speed**

---

## 🛠️ Tech Stack

- **Python**
- **OpenCV**
- **Pillow (PIL)**
- **Ollama (Moondream model)**

---

## ▶️ How It Works

1. Run the application  
2. Press **`s`** to capture a snapshot  
3. The image is analyzed by the AI model  
4. Output includes:
   - **One-line scene description**
   - **Detected objects with positions & details**
5. Annotated image is displayed  
6. Optionally ask follow-up questions  

---

## ⌨️ Controls

- **`s`** → Capture & analyze image  
- **`q`** → Quit application
